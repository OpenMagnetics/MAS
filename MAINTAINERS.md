# Maintainers

This file lists the current maintainers of the MAS specification, per
[`GOVERNANCE.md`](GOVERNANCE.md) §3.

| Name | GitHub | Area | Affiliation |
|----|----|----|----|
| Alfonso Martínez | [@AlfVII](https://github.com/AlfVII) | Schemas, reference C++ binding, governance lead during incubation | OpenMagnetics |

A formal Working Group will be constituted under the PSMA Magnetics
Committee at the start of the PSMA stewardship phase. Adding members at
that point requires WG approval per `GOVERNANCE.md` §4.

## Areas of responsibility

- **Schema editor** — single named individual responsible for spec
  text consistency and for the IEC submission package when the time
  comes. Currently: Alfonso Martínez.
- **Conformance and test vectors** — *open*; volunteers welcome.
- **Component database (`data/`)** — *open*; manufacturer-correction
  intake.
- **Reference implementation (`MAS.hpp`, MKF integration)** — Alfonso
  Martínez; coordinates with the
  [MKF](https://github.com/OpenMagnetics/MKF) maintainers.

## Becoming a maintainer

Per `GOVERNANCE.md` §4, maintainer additions during incubation require
unanimous approval of current maintainers and a track record of
substantive contributions. Open a discussion in the issue tracker if
you would like to be considered.
